Crash scripts:

1) Скрипт для No cooldown doomspire. Принцип работы прост: спамит множество бомб и локально удаляет их чтобы не лагало Исходный код скрипта:

TextLabel с переносами
Workspace.BombHolder.ChildAdded:Connect(function(class) game.Debris:AddItem(class, 0) end) local dd = 0 while true do dd += 1 game:GetService("Players").LocalPlayer.Character.Timebomb.Fire:FireServer(game.Players.LocalPlayer.Character.HumanoidRootPart.CFrame) if dd >= 20 then dd = 0 task.wait() end end

2) Скрипт для Infectious smile. Тут уже пришлось постараться чтобы сделать хоть какой анти лаг так как Bubble Messages отправляются со стороны сервера и невозможно напрямую убрать рендеринг сообщений. Однако я использовал параметр settings().Network.IncomingReplicationLag чтобы искусство вызвать задержку клиента и в конце вернуть всё чтобы все активации ClickDetector запустились. Исходный код скрипта:

settings().Network.IncomingReplicationLag = 1e9 for _ = 1, 100000 do fireclickdetector(Workspace.Map.HumanBase.Vendor.BatCollection:FindFirstChildOfClass("ClickDetector")) end settings().Network.IncomingReplicationLag = 0 task.defer(function() game:GetService("TeleportService"):TeleportToPlaceInstance(game.PlaceId, game.JobId, game.Players.LocalPlayer) end)
